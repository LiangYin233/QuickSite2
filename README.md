# QuickSite V2
> This is a Chinese readme file. Please wait for translation for readme files in other languages.

QuickSite是一个基于Node.JS的静态网站生成器。

高性能与简单。

# Better than V1
- 最高 800% 的性能提升
- 使用 markdown-meta 标记文章信息
- 最高精简 10% 的代码量

# 使用
Windows平台作为示例:

`$ qs new` 来创建一篇新博文

`$ qs build` 来构建你的网站

`$ qs clean` 用于清理Build文件夹

如果你想创建属于你自己的模板，我们将尽快更新文档。
